Getting Started

1. Project Structure

Every LimeTree project must be contained within a single .zip file. The entry point is always located at:

@start/!a.txt

2. Basic Syntax

Define labels and buttons using the internal script syntax. Example of a portal configuration:

self.label = "You are standing in a dark forest.";
self.button1.text = "Go North";
self.button1.portal = portals.north_path;

3. Using Portals

  • Portals are represented as folders in your ZIP.
  • Each folder must contain a !a.txt file for logic.
  • Use exitFromTheTree() to trigger the end-game sequence.
